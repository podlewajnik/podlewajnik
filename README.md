### podlewajnik

Podlewajnik (eng. Watering) is a self-development project with the primary goal of learning how to write automated tests using the Playwright framework. 

# Functionalities:
- Registration
- Login
- List of plants
- Adding new plants
- Editing plants
- Deleting plants
